Eagle Files Pool Controller

Outside Pool Controller - Holds Arduino Leonardo, XBee, panStamp, RTC & OLED display
Level Sensor - Transmitter for water level - holds panStamp & accelerometer
Xbee Shield - For inside pool controller which is Arduino Mega + Xbee
CT Amp - Connects to CTs measuring amps.  Uses 0-0.33 volt Current Transformers. Output is about 0-5 volts
